Descripción del Proyecto
Este proyecto consiste en un sistema básico de gestión de biblioteca desarrollado en PHP, pensado para funcionar con XAMPP.
Permite administrar libros mediante operaciones simples como agregar, listar, buscar y eliminar registros almacenados en un archivo JSON.
El objetivo es aplicar conceptos de programación, manejo de archivos, uso de clases y organización de carpetas en un proyecto real.

Cómo Ejecutar el Proyecto
Instalar XAMPP.


Copiar la carpeta biblioteca-tp dentro de:
 C:\xampp\htdocs\


Iniciar Apache desde XAMPP.


En el navegador ingresar:
 http://localhost/biblioteca-tp/
